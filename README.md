# 🧪 Test Case Prioritization System

This project is a simple web application built using Streamlit to prioritize test cases in Continuous Integration environments.

## 🚀 Features
- Upload CSV file
- Automatic prioritization
- Error handling for invalid input
- Clean UI using Streamlit

## ⚙️ How it Works
Priority Score = (failure_history * 0.5) + (coverage * 0.3) - (execution_time * 0.2)

## 📂 Files
- app.py → main app
- test_cases.csv → sample dataset
- requirements.txt → dependencies
